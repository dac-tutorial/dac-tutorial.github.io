# DAC-Tutorial 的[博客仓库](https://dac-tutorial.github.io/)

>  博客模板来自 [lemonchann 的个人博客仓库](https://github.com/lemonchann/lemonchann.github.io) ，感谢 lemonchann 的分享！
> 
>  ### 支持特性
> - Powered By Jekyll
> - 简约风格博客
> - 博客文章搜索
> - 自定义社交链接
> - 网站访客统计
> - Google Analytics 网站分析
> - Gitalk评论功能
> - 自定义关于about页面
> - 支持中文布局
> - 支持归档与标签
>
